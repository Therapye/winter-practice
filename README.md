# Механічний 7-сегментний дисплей

Цей репозиторій містить усі матеріали проєкту для створення механічного 7-сегментного дисплея. Кожен сегмент приводиться в рух окремим сервоприводом. Керування відбувається за допомогою мікроконтролера через зручний кнопковий інтерфейс.

# Структура проєкту
 `/сode` - вихідний код програми.
 `/STL files` - готові STL-файли корпусу та рухомих механізмів для 3D-друку.
 `/Schematics` - файли електричної принципової схеми, створені у KiCad.

# Апаратна частина
1) Arduino UNO
2)  PCA9685 (16-канальний I2C драйвер)
3)  7x Сервоприводів SG90
4)  2x Тактові кнопки-модулі
5)  Buck-converter
6)  2x Акумулятори 18650 (7.4V)


# Інструкція із запуску

## Крок 1, 3D-друк та збирання механіки
1. Треба роздрукувати деталі з папки `STL files`.
2. Встановіть сервоприводи у спеціальні посадкові пази в корпусі.
3. Закріпіть на валах моторів надруковані пелюстки. ВАЖЛИВО!!! фіксувати пелюстки треба лише після того, як подасте на мотори нульовий PWM-сигнал.

## Крок 2, підключення електроніки
Треба зібрати схему згідно з документацією у папці `/Schematics`. 
 **Живлення:** оскільки 7 моторів споживають багато струму, живлення на них та на мікросхему PCA9685 подається від акумуляторів через DC-DC перетворювач, і треба щоб на виході було `5V`.
 **Керування:** підключіть модуль PCA9685 до плати Arduino по  шині I2C, піни `SDA` та `SCL`. 

## Крок 3, завантаження прошивки
1. Треба відкрити папку `/code` у редакторі Visual Studio Code з розширенням PlatformIO.
2. Система автоматично підтягне потрібну бібліотеку `Adafruit PWM Servo Driver Library`.
3. Далі підключіть плату по USB та натисніть кнопку **Upload**, або комбінацією клавіш **Ctrl+Alt+U**.

## Керування пристроєм
* **Коротке натискання кнопок + / - :** ручне перемикання цифри на дисплеї на 1 крок.
* **Утримання кнопки більше 2 секунд:** запуск автоматичного режиму,  цифри почнуть перемикатися самостійно з інтервалом 1.5 секунди до досягнення ліміту 9 або 0, в залежності яку кнопку утримували.
